# Analisador de Projetos APCN - Doutorado

Aplicativo Streamlit para análise automatizada de propostas de cursos de doutorado na área 27 (Administração Pública, Contábeis, Turismo).

## Funcionalidades

- Extrai texto de arquivos .docx
- Verifica atendimento a requisitos quantitativos:
  - Corpo docente (mínimo 12 permanentes, máximo 30% colaboradores)
  - Carga horária mínima (540h)
  - Vagas e orientação
  - Produção intelectual
  - Infraestrutura
  - Autoavaliação
  - Regimento
